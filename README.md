Bem-vindo ao HackTasks! Para começar sua jornada de cibersegurança e enfrentar nossos desafios, siga as instruções abaixo:

📥 Download do Jogo
Para rodar o jogo, é necessário baixar a pasta completa do projeto, pois ele depende de arquivos externos (imagens e textos) para funcionar corretamente.

Clique no link abaixo para acessar a pasta do jogo no Google Drive:
👉 https://drive.google.com/drive/folders/148gEOPxjrqhOBcoU-q36nYDB2rWpNV44?usp=sharing

Clique no botão de Download (ícone de seta para baixo) para baixar o arquivo compactado (.zip).

Após o download, extraia o arquivo para uma pasta de sua preferência no seu computador.

🚀 Como iniciar
Após extrair a pasta, siga estes passos:

Abra a pasta extraída.

Localize o arquivo chamado Tela_inicial.exe

Dê um duplo clique nele para iniciar o jogo.

Divirta-se! Mas atenção: mantenha seu sistema seguro e não deixe os pop-ups dominarem sua tela!

⚠️ Notas Importantes
Mantenha os arquivos juntos: O arquivo .exe precisa estar na mesma pasta que os arquivos .png, capitais.txt e aves.txt. Se você mover o executável para fora desta pasta, o jogo não abrirá.

Segurança: O jogo é um ambiente controlado e seguro. Pode rodar sem preocupações!
